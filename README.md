# Sign-up-Form
The page contains the following features

1. All the fields are required to be filled if not
   it will not let you create an account
2. The name fields allows more than 3characters,
   if not its invalid. 
3. The password and confirm password inputs have
   to be the same.
4. The web page have a flexible body, it is also 
   designed to run on on mobile devices as well as 
   different sized desktops and tablets.
